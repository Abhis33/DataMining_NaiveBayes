Instructions on running the program -
######################################################
It is a simple python program and can be compiled and run using the following command -
-> python Ass1-Naive_Bayes.py
######################################################
Dependencies needed before running the program - i.e Required Libraries -
-> statistics
-> pandas
-> numpy
-> collections
-> functools

Dataset 'Adult_conv_csv - adult_data.csv' needs to be kept in same folder as code.
######################################################
If you want to provide your own test set, following things need to be done -
-> File should be in comma separated format.
-> File should include all column names.
-> Uncomment the last comments in the program, specify the path to the test set.
-> There are 2 comments one is for testing on test set and other is for accuracy metrics().
######################################################
